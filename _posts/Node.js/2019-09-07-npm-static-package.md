## npm 패키지 삭제하는 법

```
npm install module_name			# npm 모듈 설치(로컬 설치)
npm install -g module_name		# npm 모듈 설치(전역 설치)
npm update module_name			# npm 모듈 업데이트
npm uninstall module_name		# npm 모듈 삭제
```
